# Assignment 4 - FAQ Website

## 👤 Author
**Shlok Gupta**

## 📌 Overview
This project is a **Frequently Asked Questions (FAQ) website** built using **HTML, CSS, and JavaScript**.  
It allows users to click on questions to reveal answers dynamically and works responsively on any device screen size.

---

## 🛠️ Technologies Used
- HTML5  
- CSS3 (responsive design)  
- JavaScript (ES6, DOM manipulation)

---

## 📂 Files Included
| File Name      | Description |
|----------------|-------------|
| `Index.html`   | Main HTML structure of the FAQ website |
| `Style.css`    | Styling for header, FAQ items, footer, and responsive layout |
| `faq.js`       | JavaScript logic for rendering FAQ items and toggling answers |

---

## 📌 Features
- Clean and modern **FAQ layout** with a header and footer.  
- Each FAQ question has a **toggle button** (`+` / `-`) to show or hide the answer.  
- Uses **object-oriented JavaScript** (`QAItem` class) for clean code and reusable items.  
- **Responsive design** ensures it works on any screen size (desktop, tablet, mobile).  
- Smooth **fade animation** when answers are revealed.  

---

## 🚀 How to Run
1. Clone or download the repository.  
2. Ensure all three files (`Index.html`, `Style.css`, `faq.js`) are in the same folder.  
3. Open `Index.html` in a modern web browser.  
4. Click on any question to see the answer appear.

---

## 📄 Notes
- The website is fully functional without any external libraries.  
- The responsive layout uses CSS media queries to adapt for smaller devices.  
- JavaScript dynamically creates FAQ items for maintainable and scalable code.  

---

## 🖼️ Preview
**Desktop view:** Questions and answers are side-aligned with toggle icons.  
**Mobile view:** Questions stack vertically, padding and font sizes adjust for readability.
